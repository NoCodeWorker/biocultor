# AI System

## Estado

No se ha introducido una interfaz nueva de IA. La iteración SEO respeta el principio de una sola conversación definido a nivel de sistema.

## Relación con SEO para motores de IA

La estrategia implementada mejora la legibilidad para motores de respuesta mediante:

- metadata coherente
- canonicals consistentes
- structured data explícito
- páginas con intención única
- enlazado interno entre capa editorial y capa transaccional
- contenido editorial apoyado en fuentes reconocibles cuando se presentan afirmaciones de carácter técnico o científico

## Decisión

La optimización para IA no se resuelve con “páginas para bots”, sino con una arquitectura de contenido clara y verificable que pueda ser citada, resumida y entendida por sistemas de recuperación y síntesis.

En la capa editorial, “verificable” implica además:

- citar la autoridad del estudio o paper cuando el artículo apoya su argumento en evidencia externa
- enlazar o referenciar la fuente primaria cuando sea posible
- distinguir entre lo que la fuente observa y lo que Biocultor traduce como criterio de compra
